Step 2

Aufgabe: Konfetti

Die Maus Clicks des Users zählen, und dann folgendes machen:

  1. Click: Konfetti wird größer
  2. Click: Konfetti ändert die Farbe (d.h. erstmal hat es nur eine Farbe und wechselt NICHT)
  3. Click: Animation wird schneller
  4. Click: Alles geht auf den Anfangszustand und die Clicks zählen von vorn

Info

  • Hier "malen" wir mit p5.js auf einem HTML5 <canvas>
  • Dazu muss man unter Umständen ein Handbuch lesen p5.js Reference
  • Das ist Pixel Grafik im Browser. Wir erzeugen Animationen wie beim Daumenkino
  • Es gibt 3 vordefinierte Funktionen, die p5js in einer bestimmten Reihenfolge aufruft:
    1. preload zum Laden von externen Dateien
    2. setup zum einmaligen Festlegen von Parametern
    3. draw wird regelmäßig mit der aktuellen frame rate aufgerufen
    Diese Funktionen können wir in unserem Programm definieren und damit eigene Logik ausführen: 
    function prelaod() {
  // Einige Images laden
}

function setup() {
  let canvas = createCanvas(windowWidth, windowHeight)
  canvas.parent('thecanvas')
}

function draw() {
  background(40, 10)
  let c = color(Math.random() * 256, Math.random() * 256, Math.random() * 256)
  fill(c)
  ellipse(Math.random() * windowWidth, Math.random() * windowHeight, 20, 20)
}

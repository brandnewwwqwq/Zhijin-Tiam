Step 1

Aufgabe: Interaktive Glühbirne

Wenn der User mit der Maus clickt, dann soll die Glühbirne angehen, wenn sie aus war und ausgehen, wenn sie an war.

Info

  • Mit Javascript eine HTML Seite verändern.
  • Dazu muss man unter Umständen ein Handbuch lesen z.B. JavaScript and HTML DOM Reference
  • Wir wollen in die aktive Seite im Browser eingreifen mit Mitteln die die Browser für Javascript extra bereitstellen.
  • ALLES kann verändert werden: Inhalt und Style!
  • Was man dazu in Javascript braucht?
    1. Eine HTML Seite, die das Javascript Programm auch lädt und startet 
      <script type="text/javascript" src="js/ps_step_1.js" id="homework"></script>

<body onload="init()">
    2. Daten: "wir müssen uns etwas merken": 
      let hugo = "Hello World!";
    3. Funktionen: "wir müssen etwas ausführen": 
      function init() {
  console.log(hugo);
}
    4. Browser Schnittstellen: "wir müssen mit dem Browser reden": 
      let lampe = document.getElementById('lightbulb');
    5. Browser Schnittstellen: "der Browser muss uns etwas mitteilen können": 
      function init() {
  document.addEventListener('click', onMouseClick);
}

function onMouseClick() {
  console.log("Der User hat auf die Seite geklickt!");
}
    6. Browser Schnittstellen: "wir müssen Elemente im Browser verändern können": 
      lampe.setAttribute('fill', '#FFFF00')
    7. Javascript Statements: "wir brauchen so einiges an Programm Logik": 
      function onMouseClick() {
  console.log("Der User hat auf die Seite geklickt!");
  if (/* eine Variable "abfragen" */) {
    // Die Antwort ist wahr
    // Lampe einschalten
  } else {
    // Die Antwort ist falsch
    // Lampe ausschalten
  }
}
